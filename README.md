# Fast Food Tycoon 98 - Expansion Pack

Jogo de gerenciamento de fast food estilo Windows 95/98, feito em HTML/CSS/JS puro, num único arquivo.

**Alteração feita:** a limpeza das lojas agora é gratuita (custo alterado de $80 pra $0).

## Como subir no GitHub Pages

1. Crie um repositório novo no GitHub (ex: `fast-food-tycoon`).
2. Suba o arquivo `index.html` pra raiz do repositório (o nome precisa ser `index.html` pro GitHub Pages reconhecer automaticamente).
3. Vá em **Settings > Pages** no repositório.
4. Em "Source", selecione a branch `main` (ou `master`) e a pasta `/ (root)`.
5. Salva e espera uns minutinhos — o link vai aparecer ali mesmo em Settings > Pages, algo tipo:
   `https://seu-usuario.github.io/fast-food-tycoon/`

É um único HTML autocontido — sem build, sem dependências pra instalar.
